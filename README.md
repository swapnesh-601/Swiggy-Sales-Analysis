## 📌 Project Overview

This project focuses on analyzing Swiggy sales data using Python and data analytics techniques. The objective is to identify sales trends, customer behavior patterns, restaurant performance, and key business metrics to support data-driven decision-making.

The analysis includes data cleaning, exploratory data analysis (EDA), KPI tracking, and business-oriented visualizations.

---

## 🎯 Business Requirements

### Key Performance Indicators (KPIs)

- Total Sales (₹)
- Average Rating
- Average Order Value (₹)
- Ratings Count
- Total Orders

### Business Questions

- What is the total revenue generated from orders?
- What is the average customer rating?
- What is the average order value?
- Which cities contribute the highest sales?
- How do sales vary monthly and weekly?
- What is the contribution of Veg vs Non-Veg food categories?
- Which states generate the highest revenue?

## 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/your-username/Swiggy-Sales-Analysis.git
```

```bash

```

### Install Dependencies


